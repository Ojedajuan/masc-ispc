## Instalación

1. Para correr el Proyecto debemos de pararnos en la rama `frontend`:

```bash
cd frontend
```

2. Instalar las dependencias:

```bash
npm install

// ó

npm i
```

3. Y correr el servidor:

```bash
ng serve

// ó

ng s
```
